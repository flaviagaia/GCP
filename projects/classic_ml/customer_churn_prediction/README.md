# Customer Churn Prediction

## Português
Projeto de classificação binária para prever cancelamento de clientes com atributos de engajamento, suporte, billing e tenure.

Modelos comparados:
- `Logistic Regression`
- `RandomForestClassifier`

Saída:
- probabilidade de churn
- baseline selecionado por `ROC-AUC`

## English
Binary classification project for customer churn prediction using engagement, support, billing, and tenure-related features.
