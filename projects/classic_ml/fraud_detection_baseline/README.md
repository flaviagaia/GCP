# Fraud Detection Baseline

## Português
Projeto de benchmark de fraude combinando uma abordagem supervisionada e uma abordagem de detecção de anomalias.

Modelos comparados:
- `Logistic Regression` com balanceamento de classe
- `IsolationForest`

Objetivo:
- comparar performance em um problema raro
- destacar a importância de `average_precision`

## English
Fraud benchmark combining a supervised classifier and an anomaly detection baseline for rare-event modeling.
