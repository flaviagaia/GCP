# Credit Default Prediction

## Português
Projeto de classificação binária para estimar inadimplência usando atributos tabulares sintéticos com semântica de risco de crédito.

Modelos comparados:
- `Logistic Regression`
- `RandomForestClassifier`

Variáveis principais:
- estabilidade de renda
- razão dívida/renda
- utilização de crédito
- histórico de atraso
- pressão de parcelas

Saída:
- probabilidade de default
- baseline selecionado por `ROC-AUC`

## English
Binary classification project for default risk estimation using synthetic tabular data with credit-risk semantics.
